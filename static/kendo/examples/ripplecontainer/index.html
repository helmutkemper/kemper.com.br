Information

The Kendo UI RippleContainer provides the Material ink ripple effect for the Kendo UI components and is compatible only with the Sass-based Material Theme. In order to preview the example, make sure that the "Sass Material" theme is selected from the "Change Theme" menu in the top-right corner of the demo page.
